AE AI Assistant 使用说明
========================

一、安装

1. 安装 Node.js 18 或更高版本。
2. 在项目目录运行：

   npm install
   npm run install:ae

3. 完全关闭并重新打开 After Effects。
4. 在 AE 菜单打开：窗口 > 扩展 > AE AI Assistant。

二、API 设置

1. 打开插件里的 API 页面。
2. 选择供应商预设，例如 DeepSeek、Kimi、豆包、MiMo 或自定义。
3. 填入 API Key，获取模型后保存档案。
4. 保存后，主对话和生成页面都可以切换模型。

三、对话和 MD

1. 点击“新对话”可以选择一个或多个 Markdown 文件。
2. 这些 MD 只会作为当前对话的上下文，不会每次重新弹窗读取。
3. 每个对话会单独保存，归档/对话数据目录可以在历史页面设置。

四、脚本启动菜单

1. 先准备一个 Markdown 文件，里面写要启动的脚本路径，例如：

   1. 整理图层 - D:\AE脚本\整理图层.jsx
   2. 批量改名 - D:\AE脚本\批量改名.jsxbin

   也可以写成：

   [整理图层](D:\AE脚本\整理图层.jsx)

2. 在插件里点击“新对话”，选择这个 Markdown 文件作为本次对话的 MD。
3. 进入对话后，点击输入框左侧“+”，选择“脚本启动菜单”。
4. 插件不会再弹文件选择框，会直接读取当前对话已经选择的 MD。
5. AI 会在聊天里列出脚本，并询问要启动哪一个。
6. 输入数字，例如 1，点击发送，就会直接启动对应脚本。

注意：这里只会启动当前对话 MD 文件里列出的 .jsx、.jsxbin 或 .js 脚本，不会让 AI 自己临时编造脚本路径。

五、归档和保存位置

1. 打开“历史”页面。
2. 在“对话数据目录”选择保存完整对话数据的位置，建议放在 D 盘。
3. 在“对话归档目录”输入归档位置，例如：

   D:\AE AI Assistant\Archives

4. 点击“保存位置”。以后归档时会使用这里保存的位置，不会临时弹窗要求选择目录。

六、注意事项

1. AI 操作 AE 前会先生成可审查的动作预览。
2. 删除图层或关键帧需要二次确认。
3. 执行后可用 Ctrl+Z 撤销。
4. API Key 使用 Windows DPAPI 加密保存，不会写入 GitHub。
