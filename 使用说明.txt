AE AI Assistant 使用说明
========================

一、安装要求

1. Windows 10/11
2. After Effects 2025 或 2026
3. 已安装 Node.js 18 或更高版本


二、安装方法

1. 下载并解压本项目。
2. 在项目文件夹空白处点击鼠标右键，选择“在终端中打开”。
3. 依次执行：

   npm install
   npm run install:ae

4. 安装完成后，完全关闭并重新打开 After Effects。
5. 在 AE 菜单中打开：

   窗口 → 扩展 → AE AI Assistant

   部分 AE 版本可能显示为：窗口 → 扩展（旧版）。


三、首次使用

1. 打开插件左侧的“API”页面。
2. 新建 API 档案。
3. 填写 Base URL、API Key 和模型名称。
4. 勾选接口支持的功能：chat、image、video。
5. 点击“保存档案”，然后点击“测试连接”。

插件内置 OpenAI、DeepSeek、Kimi、通义千问、智谱 GLM、小米 MiMo、火山豆包和自定义接口预设。选择供应商后会自动填写基础地址，用户主要填写 API Key 并获取模型。保存后的档案可以再次点击修改。


四、主要功能

对话：使用自然语言生成 AE 操作，确认后执行。
生成：生成图片或视频素材，并自动导入 AE。
模板：使用或创建常用提示词模板。
API：管理模型供应商、密钥、模型和余额接口。
历史：查看会话、生成任务和 Token 使用量。

聊天、图片生成和视频生成页面都可以分别切换供应商与模型。


五、注意事项

1. AI 操作执行前会显示预览。
2. 删除图层或关键帧需要确认两次。
3. 执行后可按 Ctrl+Z 撤销整次操作。
4. 生成素材前请先保存 AE 工程。
5. API Key 使用 Windows 加密保存，不会上传到 GitHub。
6. 在“历史”页面点击“选择目录”，可以把完整对话归档到 D 盘或其他本地目录。
7. 归档文件为 Markdown 格式；C 盘状态文件只保留标题、摘要和文件路径。


六、插件没有显示怎么办

1. 确认已经完全重启 AE。
2. 重新执行：npm run install:ae
3. 检查安装目录：

   %APPDATA%\Adobe\CEP\extensions\com.chenyu.aeaiassistant


七、脚本启动菜单

1. 先准备一个 Markdown 文件，里面写要启动的脚本路径，例如：

   1. 整理图层 - D:\AE脚本\整理图层.jsx
   2. 批量改名 - D:\AE脚本\批量改名.jsxbin

   也可以写成：

   [整理图层](D:\AE脚本\整理图层.jsx)

2. 在对话输入框左侧点击“+”。
3. 选择“脚本启动菜单”，再选择这个 Markdown 文件。
4. 插件会列出可启动脚本，并询问要启动哪一个。
5. 输入数字，例如 1，点击发送，就会直接启动对应脚本。

注意：这里只会启动 Markdown 文件里列出的 .jsx、.jsxbin 或 .js 脚本，不会让 AI 自己临时编造脚本路径。
