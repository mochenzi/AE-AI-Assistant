AE AI Assistant 使用说明
========================

一、安装要求

1. Windows 10/11
2. After Effects 2025 或 2026
3. 已安装 Node.js 18 或更高版本


二、安装方法

1. 下载并解压本项目。
2. 在项目文件夹空白处点击鼠标右键，选择“在终端中打开”。
3. 依次执行：

   npm install
   npm run install:ae

4. 安装完成后，完全关闭并重新打开 After Effects。
5. 在 AE 菜单中打开：

   窗口 → 扩展 → AE AI Assistant

   部分 AE 版本可能显示为：窗口 → 扩展（旧版）。


三、首次使用

1. 打开插件左侧的“API”页面。
2. 新建 API 档案。
3. 填写 Base URL、API Key 和模型名称。
4. 勾选接口支持的功能：chat、image、video。
5. 点击“保存档案”，然后点击“测试连接”。


四、主要功能

对话：使用自然语言生成 AE 操作，确认后执行。
生成：生成图片或视频素材，并自动导入 AE。
模板：使用或创建常用提示词模板。
API：管理模型供应商、密钥、模型和余额接口。
历史：查看会话、生成任务和 Token 使用量。


五、注意事项

1. AI 操作执行前会显示预览。
2. 删除图层或关键帧需要确认两次。
3. 执行后可按 Ctrl+Z 撤销整次操作。
4. 生成素材前请先保存 AE 工程。
5. API Key 使用 Windows 加密保存，不会上传到 GitHub。


六、插件没有显示怎么办

1. 确认已经完全重启 AE。
2. 重新执行：npm run install:ae
3. 检查安装目录：

   %APPDATA%\Adobe\CEP\extensions\com.chenyu.aeaiassistant
